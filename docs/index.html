Names and Tokens

XML 1.0とXML 1.1のNameStartChar, NameCharは等しい。 MicroXMLのnameStartChar, nameCharは名前空間を取りのぞくために":"を持たない。 このほかに、MicroXMLはnonCharacterCodePointの除外によっていくつかのコードポイントを除外している。 この除外については、Character RangeのNoteも参照する。

HTML5ではタグ名 (tag name) はASCII alphanumericと定義される。 属性名 (attribute name) は下記以外の文字と定義される。

  • controls
    • C0 control
      • U+0000 NULLからU+001F INFORMATION SEPARATOR ONEの範囲
    • U+007F DELETEからU+009F APPLICATION PROGRAM COMMANDの範囲
  • U+0020 SPACE
  • U+0022 (")
  • U+0027 (')
  • U+003E (>)
  • U+002F (/)
  • U+003D (=)
  • noncharacters
    • U+FDD0からU+FDEFの範囲
    • U+?FFFEとU+?FFFF（?は0x00から0x10）
