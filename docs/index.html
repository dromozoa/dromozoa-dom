Names and Tokens

XML 1.0とXML 1.1のNameStartChar, NameCharは等しい。 MicroXMLのnameStartChar, nameCharは名前空間を取りのぞくために":"を持たない。 このほかに、nonCharacterCodePointの除外によっていくつかのコードポイントを除外している。
